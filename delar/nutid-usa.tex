\subsection{Dagens patentsystem i USA}

I USA innebär ett patent att innehavaren kan \emph{exkludera} andra aktörer från marknaden. Mer specifikt kan denne förhindra andra från att \emph{tillverka}, \emph{använda} eller \emph{sälja} det som skyddats. Sedan internationella patentregler infördes 1996 gäller exklusiviteten även att \emph{erbjuda att sälja} samt att \emph{importera}.
